---
## This repositiry requires Node.JS v10+ for local deployment
---
### To run Deploy Server:
    1. Run 'npm run i'
    2. Run 'npm run dev'
    3. Open http://localhost:1234

### Troubleshooting Deploy Server:
    1. Delete '/dist' directory
    2. Delete '/.cache' directory
    3. Run 'npm i'
